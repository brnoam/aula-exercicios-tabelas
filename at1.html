Funcões das Tags


● TABLE

É usada para definir tabelas

● TH

Define uma célula cabeçalho do grupo de células de sua tabela

● TR

Define uma linha numa tabela.

● TD

Cria uma coluna para inserção de conteúdos

● THEAD

É utilizado para destacar os títulos das colunas de uma tabela.

● TBODY

Estrutura todo o conteúdo da tabela

● TFOOT

Elemento para o rodapé da tabela.

● ALIGN

Usada para alinhamento

● BORDER

É responsável por adicionar bordas em qualquer elemento HTML

● CAPTION

representa o título de uma tabela

● ROWSPAN

É o número de linhas que você quer que sejam ocupadas

● COLSPAN

Permite que uma célula ocupe várias colunas, ou seja, permite mesclar colunas.

● WIDTH

Determina a largura da área de conteúdo de um elemento.
